# Relationships

Relationships can be stored as semantic metadata and exposed through introspection.

<CodeBlock>
```typescript
import { belongsTo } from '@hypequery/datasets';

const Orders = dataset('orders', {
  source: 'orders',
  dimensions: {
    customerId: dimension.string({ column: 'customer_id' }),
  },
  relationships: {
    customer: belongsTo(() => Customers, { from: 'customerId', to: 'id' }),
  },
});
```
</CodeBlock>

<Callout type="info" title="Current execution support">
  Relationship-aware query execution is not shipped in the current public API. Query execution is same-dataset only. Relationship metadata is useful for documentation, agents, and schema compatibility checks.
</Callout>
