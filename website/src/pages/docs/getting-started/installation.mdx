# Installation

Install the ClickHouse adapter with your package manager of choice.

```bash
npm install @hypequery/clickhouse
```

```bash
pnpm add @hypequery/clickhouse
```

```bash
yarn add @hypequery/clickhouse
```

## Requirements

- Node.js 18+
- A ClickHouse cluster or local instance
