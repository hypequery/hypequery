# Your first query

Define a simple query against an events table and execute it with full type
safety.

```ts
const dau = await db
  .table("events")
  .select(db.fn.countDistinct("user_id").as("dau"))
  .where("created_at", "gte", new Date())
  .execute();

console.log(dau[0].dau);
```
