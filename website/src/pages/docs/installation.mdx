# Installation

Get started using hypequery in your project by following these simple steps.

## Prerequisites

Before installing hypequery, make sure you have:

- Node.js 16 or higher
- npm or yarn package manager
- A ClickHouse instance to connect to either (cloud or self-hosted)

## Installing the Package

Install HypeQuery using npm:

```bash
npm install @hypequery/core
```

Or using yarn:

```bash
yarn add @hypequery/core
```

## Basic Setup

1. First, create a `.env` file in your project root:

```env
CLICKHOUSE_HOST=your-instance.clickhouse.cloud
CLICKHOUSE_USER=default
CLICKHOUSE_PASSWORD=your-password
CLICKHOUSE_DATABASE=default
```

2. Create a new file for your database configuration (e.g., `db.ts`):

```typescript
import { createQueryBuilder } from '@hypequery/core';

// For TypeScript users, you'll want to generate types for your schema
// We'll cover this in the next section
export const db = createQueryBuilder({
  host: process.env.CLICKHOUSE_HOST,
  username: process.env.CLICKHOUSE_USER,
  password: process.env.CLICKHOUSE_PASSWORD,
  database: process.env.CLICKHOUSE_DATABASE,
});
```

## Your First Query

Here's a simple example to get you started:

```typescript
// Import the db instance from your configuration file
import { db } from './db';

async function getRecentOrders() {
  const results = await db
    .table('orders')
    .select(['id', 'customer_name', 'amount'])
    .where('created_at', 'gt', '2024-01-01')
    .orderBy('created_at', 'DESC')
    .limit(10)
    .execute();

  console.log('Recent orders:', results);
}
```

## Type Safety (Recommended)

For the best development experience, generate TypeScript types for your ClickHouse schema:

1. Install the CLI tool globally:

```bash
npm install -g @hypequery/cli
```

2. Generate types for your schema:

```bash
hypequery generate-types
```

3. Update your configuration to use the generated types:

```typescript
import { createQueryBuilder } from '@hypequery/core';
import { Schema } from './generated/schema';

export const db = createQueryBuilder<Schema>({
  host: process.env.CLICKHOUSE_HOST,
  username: process.env.CLICKHOUSE_USER,
  password: process.env.CLICKHOUSE_PASSWORD,
  database: process.env.CLICKHOUSE_DATABASE,
});
```

Now you'll get full TypeScript support, including:
- Autocomplete for table names
- Type-safe column selection
- Validation of filter conditions
- Type checking for aggregations
