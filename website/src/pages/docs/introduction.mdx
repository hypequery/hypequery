# Introduction

hypequery is a typeScript SDK for building type-safe dashboards with ClickHouse.

## Why hypequery?

hypequery enables you to build scalable, embedded analytics experiences with Clickhouse, offering query type safety 
and features tailored to managing interactive dashboard experiences.

## Features

- **Type-safe**: Built with typescript, so you get compile-time safety for your queries
- **Extensible, Easy to use API**: Built with an extensible architecture that allows you to 
build queries in a declarative, extensible way.
- **Dashboard-specific**: Features tailored to managing interactive dashboard experiences, including global cross-filtering,
pagination support and analytics query helpers.
