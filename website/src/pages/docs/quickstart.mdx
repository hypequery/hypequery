# Quick Start

Start building a modern dashboard in under five minutes with HypeQuery, a TypeScript-first library for building type-safe dashboards with ClickHouse.

## Create Your Project

The fastest way to get started is with our CLI:

```bash
npx hypequery-create my-dashboard
cd my-dashboard
```

This will create a new project with:
- A React dashboard template
- Type-safe query builder setup
- Pre-configured TypeScript and build tools
- Example components and queries

## Connect to ClickHouse

Open `.env` and add your ClickHouse credentials:

```env
CLICKHOUSE_HOST=your-instance.clickhouse.cloud
CLICKHOUSE_USER=default
CLICKHOUSE_PASSWORD=your-password
CLICKHOUSE_DATABASE=default
```

## Generate Type Definitions

HypeQuery provides full type safety by generating TypeScript definitions from your ClickHouse schema:

```bash
npm run generate-types
```

This will create type definitions for all your tables and columns, enabling full TypeScript support in your queries.

## Your First Query

Now you can start building type-safe queries. Here's a simple example:

```typescript
import { createQueryBuilder } from '@hypequery/core';
import { IntrospectedSchema } from './generated/generated-schema';

const db = createQueryBuilder<IntrospectedSchema>({
  host: import.meta.env.CLICKHOUSE_HOST,
  username: import.meta.env.CLICKHOUSE_USER,
  password: import.meta.env.CLICKHOUSE_PASSWORD,
  database: import.meta.env.CLICKHOUSE_DATABASE,
});

// Your first query - TypeScript will autocomplete table and column names!
const results = await db
  .table('your_table')
  .select(['date', 'value'])
  .where('date', 'gt', '2024-01-01')
  .sum('value', 'total')
  .groupBy('date')
  .execute();
```

## Start the Development Server

Run your dashboard locally:

```bash
npm run dev
```

Visit `http://localhost:3000` to see your dashboard in action!
