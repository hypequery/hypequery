## Code Example

Here's a simple example of using the hypequery ClickHouse library:

```typescript
import { ClickHouseConnection, createQueryBuilder } from '@hypequery/clickhouse';

// Initialize the connection
ClickHouseConnection.initialize({
  host: 'http://localhost:8123',
  username: 'default',
  password: 'password',
  database: 'my_database'
});

// Create a query builder
const queryBuilder = createQueryBuilder();

// Build and execute a query
const results = await queryBuilder
  .from('my_table')
  .select(['id', 'name', 'value'])
  .where('value > ?', [100])
  .orderBy('name', 'asc')
  .limit(10)
  .execute();
```
