# Function: datePart()

**datePart**(`part`, `field`, `alias`?): [`SqlExpression`](/docs/reference/api/../interfaces/SqlExpression) \| [`AliasedExpression`](/docs/reference/api/../interfaces/AliasedExpression)

Extracts the specified part from a date/time value

## Parameters

### part

The part to extract (year, month, day, etc.)

`"year"` | `"quarter"` | `"month"` | `"week"` | `"day"` | `"hour"` | `"minute"` | `"second"`

### field

`string`

The field to extract from

### alias?

`string`

Optional alias for the result

## Returns

[`SqlExpression`](/docs/reference/api/../interfaces/SqlExpression) \| [`AliasedExpression`](/docs/reference/api/../interfaces/AliasedExpression)

SQL expression or aliased expression
