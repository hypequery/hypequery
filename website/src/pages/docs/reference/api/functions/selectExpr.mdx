selectExpr

# Function: selectExpr()

**selectExpr**(`sql`, `alias`?): [`SqlExpression`](/docs/reference/api/interfaces/SqlExpression) | [`AliasedExpression`](/docs/reference/api/interfaces/AliasedExpression)

Creates a raw SQL expression and optionally assigns an alias that hypequery can track throughout the query builder.

## Parameters

### sql

`string`

Raw SQL fragment to inject into the SELECT list.

### alias?

`string`

Optional alias. When provided, the returned value is an [`AliasedExpression`](/docs/reference/api/interfaces/AliasedExpression) and the alias becomes part of the result type.

## Returns

[`SqlExpression`](/docs/reference/api/interfaces/SqlExpression) | [`AliasedExpression`](/docs/reference/api/interfaces/AliasedExpression)

## Example

```ts
import { selectExpr } from '@hypequery/clickhouse';

const query = db.table('trips')
  .select([
    selectExpr('toStartOfWeek(pickup_datetime)', 'week'),
    selectExpr('count()', 'trip_count')
  ])
  .groupBy('week');
```
